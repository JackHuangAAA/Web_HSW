## 指纹模块
   > (插件名称:FINGER)
   
#### 
方法名|参数|成功|失败|备注
:---:|:---:|:---:|:---:|:---:|
open|/|如下所示|如下所示|打开设备
register|/|如下所示|如下所示|录入指纹
verify|指定序号number，按下手指|如下所示|如下所示|验证指纹
search|按下手指|如下所示|如下所示|查找指纹
delTemplateAll|/|如下所示|如下所示|删除设备所有指纹

>成功返回示例
```
{
    "code": "0000",
    "data": {
        "res": "设备名称xxxx"
    },
    "message": "指纹设备已连接"
}
```
>失败返回示例
```
{
    "code": "9999",
    "data": null,
    "message": "错误信息"
}
```  
>指纹录入成功
```
{
    "code": "0000",
    "data": {
        "number": "xxxx序号",
        "finger":"指纹特征",
        "image":"指纹图片base64加密"
    },
    "message": "录入成功"
}
```
>验证指纹成功
```
{
    "code": "0000",
    "data": {
        "number": "xxxx序号"
    },
    "message": "录入成功"
}
```
